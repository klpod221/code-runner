Code Runner API

Execute code in multiple languages through a secure, scalable, and performant RESTful API.

API Documentation GitHub 
// Execute code with a simple API call
fetch('https://code-runner.klpod221.com/api/code/run', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({
    'languageId': 'bbdef3b0-a774-4382-9a9e-b263b8a46701', // Python ID
    'code': 'print("Hello, World!")', // Your code here
    'stdin': '', // Optional input
    'isBase64Encoded': false // Optional
  })
})
.then(response => response.json())
.then(data => console.log(data));

Key Features

Code Runner API provides a simple and powerful way to execute code in various programming languages.
It is designed to be secure, scalable, and easy to integrate into your applications.

Multi-language Support

Execute code in JavaScript, Python, Java, C, C++, and more programming languages.

Secure Execution

Code runs in isolated environments to ensure security and prevent system compromise.

Test Case Support

Execute code against predefined test cases and validate outputs for assessment.

API Documentation

Comprehensive Swagger documentation for easy integration with your applications.

Authentication

JWT-based authentication system for secure access to API endpoints.

Base64 Encoding

Code and test cases are sent as Base64 encoded strings for secure transmission.

API Overview

Simple and powerful endpoints for code execution

Endpoint Description
/api/code/run Execute code in selected language
/api/code/run-tests Execute code with test cases
/api/languages List all supported languages
/api/auth/register Register a new user
/api/auth/login Authenticate a user

Getting Started

Quick start guide to use Code Runner API

Using Docker (Recommended)
  1. Clone the repository:
    git clone https://github.com/klpod221/code-runner.git
    cd code-runner
  2. Create a .env file in the project root (see .env.example for required values)
  3. Start the application:
    docker-compose up -d
  4. Access the API at http://localhost:3000 and the API documentation at http://localhost:3000/docs
